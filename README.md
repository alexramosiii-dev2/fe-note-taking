# Note Taking App (Frontend-Only)

A simple note-taking application built with vanilla CSS and JavaScript. Utilizes local storage to ensure data persistence, allowing you to store your notes even after the browser is closed.

## Description

This project is a straightforward **CRUD (Create, Read, Update, Delete)** note-taking app designed to be beginner-friendly. It serves as a great starting point for aspiring developers learning the fundamentals of frontend web development.
